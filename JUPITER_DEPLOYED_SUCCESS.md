## 🎯 NEXT STEPS TO GO LIVE

### Step 1: Setup Auto-Deploy (5 minutes, one-time)

**Option A: cPanel Git Version Control** (RECOMMENDED)
1. Login to your hosting cPanel
2. Find **"Git Version Control"** under "Files"
3. Click **"Create"** or **"Import Repository"**
4. Enter repository URL: `git@github.com:schrodercasey-lab/enterprisescanner-website.git`
5. Set repository path: `/home/yourusername/public_html/`
6. Set deployment path: `/public_html/` (or root web directory)
7. Click **"Create"** to clone repository
8. Enable **"Pull and Deploy on Push"** or **"Auto Deploy"**
9. Save settings

**Result:** Future updates = automatic! Just push to GitHub and your live site updates instantly.

---

**Option B: Manual Pull (Until Auto-Deploy Configured)**
1. In cPanel → Git Version Control
2. Find your repository
3. Click **"Manage"**
4. Click **"Pull or Deploy"** → **"Update from Remote"**
5. Confirm deployment

**Do this once now, then setup auto-deploy for future patches.**

---

### Step 2: Verify SSL/HTTPS (CRITICAL for WiFi Eyes)

**Check SSL Status:**
1. Visit: https://enterprisescanner.com
2. Verify padlock icon in browser
3. Check certificate is valid and not expired

**If SSL Not Installed:**
1. cPanel → **"SSL/TLS Status"**
2. Click **"Run AutoSSL"** (Let's Encrypt - Free)
3. Wait 2-5 minutes for certificate generation
4. Verify: https://enterprisescanner.com shows secure

**Why HTTPS is Required:**
- WiFi Eyes camera access requires HTTPS (browser security policy)
- HTTP will NOT work for camera - will show permission denied
- All modern browsers enforce this security requirement
