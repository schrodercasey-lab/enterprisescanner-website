Animation Features

IntersectionObserver API

Performant scroll detection using native browser APIs. No heavy scroll event listeners.

60 FPS Animations

GPU-accelerated transforms with requestAnimationFrame for buttery smooth animations.

Mobile Optimized

Reduced motion support and optimized for touch devices. Respects user preferences.

Repeatable Animations

Add data-scroll-repeat to re-trigger animations when elements leave and re-enter viewport.

Stagger Delays

Use data-delay="1-5" to create sequential animations for list items and grids.

Customizable

Configure threshold, speed, and animation types through JavaScript options.

Usage Examples

Basic Reveal

<!-- Simple fade up animation --> <div data-scroll-reveal="fade-up"> Your content here </div>

Staggered Animation

<!-- Staggered delays for sequential effect --> <div data-scroll-reveal="fade-up" data-delay="1">First</div> <div data-scroll-reveal="fade-up" data-delay="2">Second</div> <div data-scroll-reveal="fade-up" data-delay="3">Third</div>

Parallax Effect

<!-- Parallax scrolling --> <div data-parallax="0.5"> Moves at half the scroll speed </div>

Section Progress

<!-- Add section navigation dots --> <section data-section="features" data-section-label="Features"> Your section content </section>

JavaScript Initialization

// Auto-initializes on DOMContentLoaded // Or manually initialize: const scrollAnimations = new ScrollAnimations({ enableParallax: true, enableReveal: true, enableProgress: true, enableSmoothScroll: true, revealThreshold: 0.15, parallaxSpeed: 0.5 });
