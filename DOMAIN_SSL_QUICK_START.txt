🎯 STEP 1: CONFIGURE DNS (5 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📍 Where: Log into your domain registrar
   (Where you bought enterprisescanner.com)
   Examples: GoDaddy, Namecheap, Google Domains

📝 Add these DNS records:

   Type  | Name | Value
   ─────────────────────────────
   A     | @    | 134.199.147.45
   A     | www  | 134.199.147.45

💡 Quick guides:
   • GoDaddy: My Products → Domain → Manage DNS → Add Record
   • Namecheap: Domain List → Manage → Advanced DNS → Add Record
   • Google Domains: My Domains → DNS → Custom Records

⏳ Wait 5-15 minutes for DNS to propagate

✅ Test DNS (run on YOUR computer):
   
   nslookup enterprisescanner.com
   
   Should show: 134.199.147.45

🔍 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Issue: DNS doesn't resolve
Solution:
   • Check DNS records at registrar (correct IP?)
   • Wait longer (can take up to 48 hours)
   • Test: nslookup enterprisescanner.com

Issue: SSL certificate fails
Solution:
   • Verify DNS points to 134.199.147.45 first
   • Check port 80 is open: curl http://134.199.147.45
   • Ensure no rate limit (max 5 certs per week)

Issue: HTTPS doesn't work
Solution:
   • Check if port 443 is open: netstat -tuln | grep 443
   • View logs: docker logs enterprisescanner_nginx
   • Check certificate: certbot certificates
